# ☕ Yaggo's Café - Landing Page

Uma landing page moderna, responsiva e elegante desenvolvida para uma cafeteria premium. O projeto apresenta o cardápio, depoimentos de clientes e a localização física da loja.


##  Tecnologias Utilizadas

* **HTML5:** Estrutura semântica do site.
* **CSS3:** Estilização com foco em cores escuras e douradas (tema premium).
* **Fontes Google Fonts:** Para uma tipografia legível e sofisticada.
* **Font Awesome:** Para ícones de redes sociais e navegação.
* **Google Maps API/Iframe:** Para exibição da localização real no rodapé.

##  Funcionalidades

* **Seção Sobre Nós:** Detalhes sobre o processamento especializado dos grãos.
* **Menu Interativo:** Exibição de preços e fotos dos cafés (Café Coado, Arábica, Cappuccino, Dalgona, etc.).
* **Depoimentos:** Área dedicada ao feedback dos clientes com sistema de avaliação por estrelas.
* **Localização:** Mapa interativo integrado para facilitar a chegada dos clientes.
* **Responsividade:** O site foi pensado para ser visualizado tanto em desktops quanto em dispositivos móveis.

##  Demonstração

O site utiliza uma paleta de cores voltada para o **Black & Gold**, transmitindo uma sensação de exclusividade e qualidade premium.

* **Destaque do Cardápio:** Preços promocionais e fotos atrativas.
* **Redes Sociais:** Links integrados no rodapé para Instagram, Facebook e GitHub.
